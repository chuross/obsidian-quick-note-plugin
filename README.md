# Obsidian Quick Note Plugin

Obsidian用のTwitterライクなクイック投稿プラグインです。入力した内容は自動的にデイリーノートに保存され、`- HH:mm 内容` の形式でタイムスタンプ付きで管理されます。

## 機能

- **統合ビュー**: 投稿画面とタイムラインが一体化したビュー。
- **添付ファイル**: 画像などのファイルを簡単に添付・埋め込み可能。
- **タイムライン**: 過去7日間の投稿を新しい順に表示。
- **モバイル対応**: iOS/Android の Obsidian アプリでもネイティブに近い操作感。

## インストール手順

1. このリポジトリをクローンまたはダウンロードし、`obsidian-quick-note-plugin` ディレクトリを Obsidian の Vault 内 `.obsidian/plugins/` に配置します。
2. Obsidian を再起動し、**Settings → Community plugins** で **Quick Note** を有効化します。

## 使い方

- **ビューを開く**: リボンの鉛筆アイコンをクリックします。
  - **PC**: 右サイドバーに表示されます。
  - **モバイル**: 全画面で表示されます。
- **投稿**: テキストエリアに入力し、`Note it` ボタンをクリック（または `Enter`、改行は `Shift+Enter`）。
- **添付**: クリップアイコン（📎）でファイルを選択して添付できます。
- **タイムライン**: 下部に直近7日間の投稿が表示されます。

## 設定

プラグイン設定画面で以下をカスタマイズできます。
- **日付フォーマット**（例: `YYYY-MM-DD`）
- **タイムスタンプフォーマット**（例: `HH:mm`）
- **挿入位置**: ファイル末尾、先頭、または指定見出しの下

## 開発

```bash
npm install
npm run dev   # 開発サーバー（watch）
npm run build # ビルド
```
