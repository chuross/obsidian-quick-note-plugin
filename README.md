# Obsidian Quick Note Plugin

Obsidian用のTwitterライクなクイック投稿プラグインです。入力した内容は自動的にデイリーノートに保存され、`- HH:mm 内容` の形式でタイムスタンプ付きで管理されます。

## 機能

- **クイックノート**: コマンドまたはリボンアイコンから素早くノートを作成し、デイリーノートに追加します。
- **タイムラインビュー**: 過去のノートをタイムライン形式で閲覧できます。
- **モバイル対応**: iOS/Android の Obsidian アプリでも動作します。

## インストール手順

1. このリポジトリをクローンまたはダウンロードし、`obsidian-quick-note-plugin` ディレクトリを Obsidian の Vault 内 `.obsidian/plugins/` に配置します。
2. Obsidian を再起動し、**Settings → Community plugins** で **Quick Note** を有効化します。
3. 必要に応じて **Settings → Plugin options → Quick Note** で設定を調整します（日付フォーマット、タイムスタンプフォーマット、挿入位置など）。

## 使い方

- **クイックノートを開く**: コマンドパレットで `Open Quick Note` を実行するか、リボンの鉛筆アイコンをクリックします。
- **ノートを入力**: テキストエリアに最大 280 文字まで入力できます。文字数カウントがリアルタイムで表示され、上限を超えると警告が出ます。
- **投稿**: `Submit` ボタンをタップすると、現在の日付のデイリーノートに `- HH:mm 内容` 形式で追記されます。
- **設定**: プラグイン設定画面で以下をカスタマイズできます。
  - **日付フォーマット**（例: `YYYY-MM-DD`）
  - **タイムスタンプフォーマット**（例: `HH:mm`）
  - **挿入位置**: ファイル末尾、先頭、または指定見出しの下

## タイムラインビュー

1. コマンドパレットで `Open Quick Note Timeline` を実行します。
2. 直近 7 日間の投稿が時系列で表示されます。各エントリはタップすると該当するデイリーノートの位置にジャンプします。

## モバイルでの利用

- UI は自動的にレスポンシブデザインに切り替わり、タップ領域が拡大されます。
- iOS/Android の Obsidian アプリでも同様にコマンドパレットまたはリボンアイコンからクイックノートを使用できます。

## 開発

```bash
npm install
npm run dev   # 開発サーバー（watch）
npm run build # ビルド
```

プラグインのコードは `src/` ディレクトリ以下にあります。変更後は `npm run build` でビルドを実行し、エラーが出ないことを確認してください。
